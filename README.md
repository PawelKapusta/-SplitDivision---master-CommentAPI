# -SplitDivision---master-CommentAPI

### Running the development server.

### To install

```bash
    npm install
```

### And run:

```bash
    npm run start
```

### To run project

#### Build docker image

```bash
    docker build -t master-comment-api .
```

#### Run a container on port 5004

```bash
    docker run -p 5004:5004 master-comment-api
```
